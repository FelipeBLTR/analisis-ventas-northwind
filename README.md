#  Análisis de Ventas – Northwind

##  Objetivo

Analizar el rendimiento de ventas para identificar los productos, categorías, clientes y mercados con mayor impacto en los ingresos del negocio.

---

##  Herramientas utilizadas

* SQL (consultas y análisis de datos)
* Power BI (visualización y dashboard)
* Base de datos Northwind

---

##  Análisis realizado

Se desarrollaron consultas SQL para responder preguntas clave del negocio:

* Ventas totales del negocio
* Ventas por país
* Top clientes por ingresos
* Top productos más vendidos
* Ventas por categoría
* Evolución de ventas por mes
* Producto más vendido por categoría (uso de CTE y RANK)
* Clientes con ventas superiores al promedio

---

##  Dashboard

Se creó un dashboard interactivo en Power BI que incluye:

* KPI de ventas totales
* Ventas por país
* Ventas por categoría
* Ventas por producto
* Evolución de ventas en el tiempo
* Segmentadores por categoría y fecha

---

##  Principales insights

* La categoría **Meat/Poultry** es la que genera mayores ingresos (~2.3M).
* Existe una alta dependencia de un solo producto dentro de esta categoría (**Thüringer Rostbratwurst**).
* Las ventas están concentradas geográficamente, con **Estados Unidos** como principal mercado, seguido por Francia.
